# 🚐 BiyaHero Admin Console
**Streamlined Fleet Management & Dispatching for Regional Transit.**

BiyaHero is a JavaFX-based management solution tailored for transport operators. It provides a centralized dashboard to oversee drivers, manage van rotations, and generate comprehensive financial and passenger reports.

---

* **MySQL 8.0+**
