# BiyaHero: Transport Management System

BiyaHero is a modern, JavaFX-based Transport Management System designed for van transport operators. It streamlines trip tracking, passenger bookings, and fleet management with a high-end, user-centric interface.

## 📋 Features

### 1. Live Trip Tracking
- Real-time progress bars for active trips.
- Automated route status updates (Scheduled -> En Route -> Completed).
- Quick-glance occupancy metrics on the dashboard.

### 2. Advanced Booking System
- **Interactive Seat Map:** Visual representation of van occupancy.
- **Segment Logic:** Capacity management that tracks which specific legs of a trip a seat is occupied.
- **Passenger Management:** Full CRUD operations for passenger records linked to specific trips.

### 3. Fleet & Personnel Management
- **Vans:** Track plate numbers, models, and maintenance status.
- **Drivers:** Manage driver licensing, contact info, and current trip assignments.


## 🛠 Tech Stack
- **Language:** Java 17
- **UI Framework:** JavaFX (FXML + CSS)
- **Database:** MySQL
- **Architecture:** Model-View-Controller (MVC) + Service Layer
- **Build Tool:** Maven
